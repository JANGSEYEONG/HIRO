## Getting Started

```
yarn install
yarn dev
```

## 🛠 기술 스택

- **Language:** TypeScript
- **Library:** Next.js 15
- **CSS:** Tailwind CSS + Shadcn/ui
- **State Management:** TanStack Query / Zustand
- **HTTP Client:** Rest API, Axios
- **Package Manager:** Yarn
- **Linting:** ESLint, Prettier
